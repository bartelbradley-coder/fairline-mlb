Pick Card Design Options

Three approaches for displaying ML + Run Line + Over/Under picks — same game, same data, different layout.

Option A
Inline Secondary Markets
One card per game. ML pick is the primary action. RL and O/U appear as compact "also-bet" rows below the card — only shown when edge ≥ 3%. Keeps the single-scroll feed feel.
